Essential shortcuts only | All shortcuts

Navigation

  • F11 Full screen
  • Alt + Left Arrow
    or Backspace Back
  • Alt + Right Arrow
    or Shift + Backspace Forward
  • Alt + Home Home
  • F5
    or Ctrl + R Reload
  • Ctrl + F5
    or Ctrl + Shift + R Reload and override cache
  • Esc Stop

Current Page

  • Ctrl + Minus Sign
    or Ctrl + Mouse Wheel Decrease text size
  • Ctrl + Plus Sign
    or Ctrl + Mouse Wheel Increase text size
  • Ctrl + 0 Restore text size

Search

  • Ctrl + F Find text
  • F3
    or Ctrl + G Find again
  • Shift + F3 Find previous
  • ' (Apostrophe) Find link as you type
  • / (Slash) Find text as you type

Windows and Tabs

  • Ctrl + T New tab
  • Ctrl + W
    or Ctrl + F4 Close tab
  • Ctrl + Tab
    or Ctrl + Page Down Next tab
  • Ctrl + Shift + Tab
    or Ctrl + Page Up Previous tab
  • Ctrl + (1-8) Select tab (1 to 8)
  • Ctrl + 9 Select last tab
  • Ctrl + Shift + T Restore recently closed tab

Tools

  • Ctrl + J Downloads
  • Ctrl + H History
  • Ctrl + B
    or Ctrl + I Bookmarks

Miscellaneous

  • Ctrl + Enter Complete .com address
  • Shift + Enter Complete .net address
  • Ctrl + Shift + Enter Complete .org address
  • Del Delete selected autocomplete entry
  • Alt + D
    or F6
    or Ctrl + L Select location bar
