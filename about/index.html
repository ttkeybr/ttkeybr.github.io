About

Using keybr.com

You can customize the following options:

  1. Switch input language and keyboard layout, if you use an unconventional one
  2. Switch text source to random text, custom text or web site
  3. Show chart with the history of your typing speed to see how you have been progressing over time

These options are available by clicking the corresponding labels at the top of the screen.

You can also explore meaning of a user interface element by hovering mouse cursor over it, this will reveal a tooltip with additional information.

Input Language and Keyboard Layout

keybr.com supports English, German, Spanish, Italian, Portuguese, French and Russian languages.
It also supports Dvorak and Colemak keyboard layouts for English language. There are not recognized automatically, so you will need to switch layout by hand.

Text Source

You can choose from random text, custom text and any of the web site or blog.

keybr.com generates random text which seems much like a natural language, but it isn't. A typical random string would look like "eventeral ful parts that ourse toody plan though uncipitabl world shally object ves into and were prote the senting thous binduring imployal and from the delphianam and they due see hight was take" for English language. The generated text does not contain capital letters, numbers and punctuation characters. The advantage of the random text is that it can be generated from letters of the keys of Caps Lock, Tab or Shift row only. For example, keybr.com can generate mostly readable text using letters of the Caps Lock row only. You can choose from different levels, it lets you learn keys row by row.

With custom text you can paste any text you want in keybr and type it out.

You can also specify URL of a web site or blog whose text you wish to type. This probably is the coolest feature of keybr.com — you can read your favorite blog and improve your typing speed at the same time. Please choose simple sites with content, but not sites overloaded with images and multimedia.

When typing, you can skip to the next phrase by pressing Ctrl - Backspace shortcut key. This will abandon the current phrase completely ignoring its statistic.

Typing Speed Chart

It shows how your typing speed changes over time.

keybr.com keeps your typing statistics on the server. But only when you are a registered user.

Happy typing!
