Keyboard shortcuts

Novice computer users often don't realize that using a mouse is not as easy, fast and comfortable as using keyboard when controlling a computer. Although using a mouse seems more intuitive at first, the keyboard still wins when you learn how to use it properly. Most of the commands available in a program's menu can be executed by pressing just a few keyboard keys. For example, while you read this article in your browser, hold down the CTRL key, then press the D key. This will display the Add Bookmark window. You can press the ENTER key to accept it, or press ESC to dismiss it. You have just invoked a shortcut, which is a combination of keys that perform a predefined function. Shortcuts can be significant time-savers, and offer a method of executing commands that process quicker than what a mouse may offer.

We have collected shortcuts for some of the most frequently used software for you to learn and use. Spend few minutes remembering them now, so you can save your time later.
